# Teslo API

1. Clonar el proyecto

2. Instalar dependencias
```bash
yarn install
```

3. clonar el archvio ```env.template``` y renombrarlo a ```.env```

4. Configurar las variables de entorno

5. Levantar la base de datos
```bash
docker compose up -d
```

6.- Levantar el modo de desarollo
```bash
yarn run start:dev
```
